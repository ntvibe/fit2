# Fit2 - Workout Planner App

Progressive Web App for fitness tracking with local storage.

## Development

Open `index.html` directly in a browser, or serve locally:

```bash
npx serve app
```

## Deployment

Deploy the `app/` folder to GitHub Pages:

1. Go to repository Settings → Pages
2. Source: Deploy from `main` / `master` branch
3. Folder: `/ (root)` - move contents of `app/` to root, or configure source to `/app`

## Features

- ✅ Create workout plans
- ✅ Add exercises with sets/reps
- ✅ localStorage persistence
- ✅ PWA (installable)
- ✅ Offline support (service worker)
- ✅ Works on GitHub Pages

## Tech Stack

- Vanilla HTML/CSS/JS
- No build step required
- localStorage for data
- Service Worker for offline
